# Privacy Policy for GitHub Tabs

Last updated: March 20, 2026

GitHub Tabs does not collect, store, transmit, sell, or share personal information.

## What the extension does

GitHub Tabs looks at your open browser tabs so it can:

- find tabs that belong to GitHub
- move GitHub tabs into the current window
- remove duplicate GitHub tabs
- sort GitHub tabs by URL

This processing happens locally in your browser.

## Data collection

GitHub Tabs does not collect any personal data.

GitHub Tabs does not send any data to external servers.

GitHub Tabs does not use analytics, tracking, advertising, cookies, or third-party services.

GitHub Tabs does not store your browsing data outside the current browser session.

## Permissions

GitHub Tabs uses the `tabs` permission only to inspect and organize your open GitHub tabs.

## Data sharing

GitHub Tabs does not sell, transfer, or share user data with any third party.

## Contact

If you have questions about this privacy policy, contact the repository owner through the GitHub repository for this project.
